Frequently asked questions

Ubitech is an open-source library of interactive components built using bootstrap classes including buttons, dropdowns, modals, navbars, and more.

Ubitech is first conceptualized and designed using the Figma software so everything you see in the library has a design equivalent in our Figma file.

The main difference is that the core components from Ubitech are open source under the MIT license, whereas UI is a paid product. Another difference is that Ubitech relies on smaller and standalone components, whereas UI offers sections of pages.
However, we actually recommend using both Ubitech, Ubitech Pro, and even UI as there is no technical reason stopping you from using the best of two worlds.
